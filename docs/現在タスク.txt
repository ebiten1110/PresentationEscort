Phase 5 作業順
Phase 5-1：Raspberry Pi起動確認

やること：

1. Raspberry Pi OSを起動
2. Wi-Fi接続
3. ターミナルを開く
4. Python確認
5. USBカメラ認識確認

確認コマンド：

python3 --version
lsusb

lsusb でBUFFALOやWeb Cameraっぽい表示が出ればOKです。

Phase 5-2：必要パッケージ導入

Raspberry Piで実行します。

sudo apt update
sudo apt install -y python3-opencv python3-pip v4l-utils

カメラ確認用：

v4l2-ctl --list-devices

ここで /dev/video0 が出れば、Webカメラが認識されています。
